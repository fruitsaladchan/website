Software I Use

This is a list of all the software i use. Most of the software is foss can be installed thorugh your package manager.

I like having vim-like bindings and perfer cli apps over bloated gui apps.

Basic

  • Operating System: Arch Linux

    An independently developed, lightweight, and flexible Linux distribution for advanced users.

  • Terminal Emulator: Kitty

    A fast, feature-rich, GPU-accelerated terminal emulator with support for images and more.

  • Shell: Zsh

    A powerful and customizable shell with advanced scripting capabilities, enhancing the command-line experience.

  • Window Manager: Hyprland

    Hyprland is a dynamic tiling window manager based on the wayland compositor.I recently just switched to Hyprland after using i3wm and am still trying it out.

  • Text Editor: Vim

    A highly configurable and efficient text editor, known for its modal interface and extensibility.

utilities

  • PDF reader: zathura

    A minimalist and highly customizable PDF viewer with vi-like keybindings, ensuring a smooth reading experience

  • Browser: firefox

    Firefox with the arkenfox user.js. A custom user.js like this one removes all the trash spyware and allows you to customize the browser to be actually private.

  • Video Player: MPV

    A free and open-source media player known for its customization and high-quality playback written in lua.

  • File manager: Ranger

    A console-based file manager with a user-friendly and efficient interface for navigating and managing files.

  • Qbittorrent: Torrent client

    qBittorrent is an open-source and easy-to-use torrent client that allows you to download and manage torrents efficiently with a clean and straightforward interface

  • Music player: Spotify with spicetify

    Spicetify is a tool used to customize the Spotify client's appearance and features

  • Note taker: Obsidian

    Obsidian is a powerful note-taking and knowledge management tool that uses a plain-text Markdown format for easy organization and accessibility
